> **A Sizzling STEM Adventure in Measurements and Fractions by James S.**

Welcome to **Grill & Measure**, an interactive, single-page educational web application designed for 4th–6th graders! This project blends the mouth-watering fun of cooking burgers with the crucial math and science concepts of fractions, measurements, and heat transfer. 

Watch the math sizzle on a **custom 3D virtual grill** powered by Three.js! 🥩🔥

---

## 🚀 Features

* **🎮 Interactive 3D Grill:** Built with `Three.js`, students can rotate, zoom, and pan around a 3D grill.
* **🧮 Visualizing Fractions:** Click interactive buttons to magically divide a 1 lb "meat block" into halves (½), thirds (⅓), and quarters (¼) on the grill.
* **🧠 Auto-Graded Math Challenges:** Built-in text inputs test kids on real-world cooking math (e.g., converting pounds to ounces).
* **🧪 STEM Integrated:** Teaches the science of cooking (Conduction vs. Convection) alongside math.
* **📱 Responsive & Kid-Friendly UI:** Warm colors, fun emojis, easy-to-read typography, and smooth-scrolling navigation.
* **⚡ Zero Build Steps:** Everything is contained in a single `index.html` file. No `npm install` required!

## 🛠️ Tech Stack

* **Frontend:** HTML5, CSS3, Vanilla JavaScript
* **3D Graphics:** [Three.js](https://threejs.org/) (Loaded via CDN)
* **Camera Controls:** `OrbitControls.js` (Allows users to drag and zoom the 3D scene)
